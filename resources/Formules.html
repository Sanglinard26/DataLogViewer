Formules

Nouvelle formule :

Pour créer une formule à partir des variables disponibles dans le log il faut aller dans le menu "Formules" puis "Nouvelle"

Une boite de dialogue pour l'édition de celle-ci :

/!\ Il est nécéssaire d'encadrer le nom de chaque variable issue du log par le caractère # /!\.
Pour les utilisateurs de PcsLab il y a la possiblité de créer des formules à partir des paramètres de calibration.

Voici quelques exemples de formules :
  • #RPM(tr/min)#/120
  • 950 => En mettant un unique nombre on définit une formule qui aura la même valeur tout au long du log (typiquement un seuil fixe)
  • TABLE2D{Avance base,RPM(tr/min),Intake_manifold_P(mbar)} => Pour chaque point du log l'avance de base sera recalculée à partir de la calibration chargée.

Gestion des formules :

Pour accéder au gestionnaire de formule il faut passer le menu "Formules", ensuite "Gestionnaire". Au sein de cette fenêtre il sera possible de modifier une formule existente ou éventuellement d'en supprimer.

La sauvegarde des formules est intégrée Ã  l'enregistrement du fichier de configuration.
