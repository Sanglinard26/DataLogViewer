Ouverture d'un fichier *.map

Clicker sur le bouton situé dans la barre d'outils

Une fois le fichier chargé, un arbre avec une structure équivalente a PcsLab s'affiche (dans le cas ou le fichier *.mdb est bien présent dans le meme répertoire que le fichier *.map).
Il est possiblie d'ouvrir plusieurs fichier map en même temps.
