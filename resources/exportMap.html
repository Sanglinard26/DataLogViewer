Export fichier map

Dans le cas où le fichier map aurait été modifié il est possible de le sauvegarder.
Pour cela il faut faire un click droit sur le nom du fichier ou d'une variable puis "Enregistrer fichier Map" comme illustré ci-dessous :

Mise en garde : Il peut arriver que certains paramètres soient mal lus à l'ouverture du fichier map.
Si tel est le cas, le fichier sauvegardé sera alteré...
Des évolutions arriveront plus tard pour gagner en robustesse.
