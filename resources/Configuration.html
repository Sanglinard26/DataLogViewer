Configuration

Le fichier de configuration (format xml) regroupe trois choses :

  • l'ensemble des propriés des graphiques
  • les formules
  • les conditions
Ouverture d'un fichier de configuration (ou via Fichier => Ouvrir configuration). Il est existe deux formats : *.cgf et *.xml. Le premier sera amené à disparaitre prochainement.

L'enregistrement d'un fichier de configuration se fait avec le menu : Fichier => Enregistrer configuration.
